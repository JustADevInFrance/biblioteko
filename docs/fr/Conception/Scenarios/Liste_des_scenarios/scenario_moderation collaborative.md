Acteurs :

Bibliothécaires (plusieurs)

IA d’aide à la décision

Déroulement :

Plusieurs bibliothécaires reçoivent une notification pour une œuvre déposée.

L’IA analyse le texte pour détecter les œuvres protégées (auteurs encore vivants, mentions légales, etc.).

L’interface TAL affiche le score de risque et suggère une catégorie.

Le bibliothécaire peut :

Valider,

Rejeter,

Demander des précisions.

Le résultat est versionné dans Git pour traçabilité.
