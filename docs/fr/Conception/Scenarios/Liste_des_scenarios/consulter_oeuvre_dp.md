## Consulter une œuvre du domaine public ##

**Description** :
Le membre accède à une œuvre libre de droits pour la lire, la télécharger ou la copier librement.

**Pré-requis** :

- Être membre connecté.

- L’œuvre est dans le fond_commun.

**Contenu du scénario** :

1. Le membre ouvre la rubrique **Fond commun**.

2. Il recherche une œuvre libre de droits dans le catalogue.

3. Il sélectionne l’œuvre.

4. L’application affiche les métadonnées (titre, auteur, date, catégorie, etc.).

5. Le membre choisit d’ouvrir ou de télécharger l’œuvre sans restriction.
