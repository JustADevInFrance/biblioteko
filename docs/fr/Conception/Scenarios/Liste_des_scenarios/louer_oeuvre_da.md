## Louer une oeuvre sous droits d'auteur ##

**Description :**  
Le membre emprunte temporairement une œuvre protégée pour une durée limitée.  

**Pré-requis :**  
- Être membre connecté.  
- L’œuvre est disponible et non expirée.  

**Contenu du scénario :**  
1. Le membre consulte le catalogue.  
2. Il sélectionne une œuvre sous droits.  
3. Il clique sur **Louer**.  
4. L’application chiffre l’œuvre avec la clé du membre.  
5. Le fichier est stocké dans le répertoire **emprunts**.  
6. Après deux semaines, l’application supprime automatiquement le fichier.
