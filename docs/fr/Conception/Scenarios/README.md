# Liste des scénarios #

- __Scénario pour [installer l'application](./Liste_des_scenarios/installer_appli.md)__ : Un utilisateur souhaite installer l'application sur son appareil pour accéder à la bibliothèque numérique décentralisée.

- __Scénario pour [devenir membre](./Liste_des_scenarios/devenir_membre.md)__ : Un utilisateur souhaite devenir membre de la bibliothèque numérique afin de pouvoir emprunter, louer et proposer des oeuvres.

- __Scénario pour [se connexter](./Liste_des_scenarios/se_connecter.md)__ : Un utilisateur souhaite se connecter à l'application afin d'accéder à son espace personnel et à ses oeuvres.


- __Scénario pour [devenir bibliothécaire](./Liste_des_scenarios/devenir_bibliothecaire.md)__ : Un membre souhaite obtenir le rôle de bibliothécaire afin de participer à la modération et à la gestion des oeuvres.

- __Scénario pour [déposer une oeuvre](./Liste_des_scenarios/deposer_oeuvre.md)__ : Un membre a numérisé une oeuvre et souhaite le partager avec la bibliothèque pour enrichir son fond. Selon le format du fichier (Markdown ou PDF), l'applicarion adapte le traitement avant de soummetre l'oeuvre à la modération.

- __Scénario pour [consulter les informations concernant une oeuvre](./Liste_des_scenarios/consulter_infos_oeuvre.md)__ : Le membre ou le bibliothécaire consulte les métadonnées d’une œuvre pour en connaître les détails.

- __Scénario pour [remplir les informations d'une oeuvre](./Liste_des_scenarios/remplir_infos_oeuvre.md)__ : Lorsqu’un membre soumet une œuvre, il renseigne les informations descriptives nécessaires à son identification et à sa classification.

- __Scénario pour [modifier les informations d'une oeuvre](./Liste_des_scenarios/modifier_infos_oeuvre.md)__ : Le bibliothécaire met à jour ou corrige les métadonnées d’une œuvre (pendant ou après la modération).

- __Scénario pour [consulter une oeuvre du domaine public](./Liste_des_scenarios/consulter_oeuvre_dp.md)__ : Le membre accède à une œuvre libre de droits pour la lire, la télécharger ou la copier librement.

- __Scénario pour [diffuser une oeuvre libre de droits](./Liste_des_scenarios/diffusion_oeuvre_libre.md)__ : Lorsqu’une œuvre devient libre, elle est automatiquement partagée avec les membres disposant d’espace disque partagé. 

- __Scénario pour [louer une oeuvre soumise aux droits d'auteurs](./Liste_des_scenarios/louer_oeuvre_da.md)__ : Le membre emprunte temporairement une œuvre protégée pour une durée limitée. 

- __Scénario pour [passer une oeuvre dans le domaine public](./Liste_des_scenarios/oeuvre_to_public.md)__ : Le système détecte qu’une œuvre protégée a atteint la fin de sa période de protection et devient libre de droits.  

- __Scénario pour [mettre à jour l'index des oeuvres](./Liste_des_scenarios/maj_index_oeuvres.md)__ : L’application met à jour automatiquement l’index général à chaque ajout, suppression ou modification d’une œuvre.  

- __Scénario pour [consulter les rubriques](./Liste_des_scenarios/consultation_rubriques.md)__ : Le membre explore les différentes rubriques de la bibliothèque pour accéder aux œuvres selon leur statut.  

- __Scénario pour [consulter la rubrique *fond commun*](./Liste_des_scenarios/consultation_fond_commun.md)__ : Le membre consulte la rubrique contenant les œuvres libres de droits proposées par l’association.
