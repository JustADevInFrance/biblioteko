# Liste des scénarios possibles #

- __Scénario pour [installer l'application]()__ : décrit le processus d'installation de l'application sur différents.

- __Scénario pour [devenir membre]()__ : Une personne souhaite devenir membre.

- __Scénario pour [devenir bibliothécaire]()__ : Un membre demande à devenir bibliothécaire.

- __Scénario pour [accéder à la liste des oeuvres]()__ : L'utilisateur demande à voir la liste des oeuvres.

- __Scénario pour [déposer une oeuvre numérisée]()__ : Un membre a numérisé une oeuvre et souhaite la partager avec la bibliothèque pour enrichir son fond.
