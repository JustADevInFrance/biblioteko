# Glossaire métier #

<span style="background: blue">Concept</span> : Mot portant une signification pour un projet.

<span style="background: darkorange">Propriété</span> : qualité, durée, etc.

<span style="background: green">Action</span> : transformation de l'état.

<span style="background: darkred">Nom propre</span> : Identifiant unique d'une chose ou d'une personne.

### A ###

- <span style="background: green">Assurer la modération</span> : action des bibliothécaires visant à contrôler et valider une oeuvre proposée.

- <span style="background: blue">a_moderer</span> : répertoire contenant les ouevres proposées par les membres en attente de validation.

- <span style="background: blue">Articles</span> : oeuvres textuelles de type journalistique ou scientifique.

- <span style="background: green">Accès gratuit</span> : possibilité pour un membre de consulter une oeuvre du domaine public sans contrepartie financière.

### B ###

- <span style="background: blue">Bibliothèque numérique décentralisée</span> : application numérique qui fonctionne comme une médiathèque, avec un stockage des oeuvres réparti entre les membres.

- <span style="background: blue">Bibliothécaires</span> : membres de l'application responsables de la vérification, de l'enrichissement des métadonnées et de la validation ou du rejet des oeuvres.

- <span style="background: blue">BD</span> : type de livre (bande dessinée).

### C ###

- <span style="background: blue">Catégorie d'oeuvres</span> : regroupement thématique ou typologique (ex. Livre, Musique, Vidéo, Article).

- <span style="background: darkred">CultureDiffusion</span> : nom de l'association et de l'application.s

- <span style="background: blue">Clé du membre</span> : donnée chiffrée permettant de sécuriser l'accès au oeuvres empruntées.

- <span style="background: green">Classification des oeuvres</span> : action d'organiser les oeuvres par catégories et sous-catégories.

### D ###

- <span style="background: blue">Domaine public</span> : statut juridique d'une oeuvre dont les droits ont expirés, rendant son usage libre.

- <span style="background: green">Diffuser automatiquement</span> : action de rendre accessible une oeuvre devenue libre de droit à tous les membres disposant d'espace disque partagé.

- <span style="background: green">Déposer une oeuvre</span> : action d'un membre consistant à proposer un fichier à la bibliothèque.

- <span style="background: darkorange">Deux semaines (période)</span> : durée maximale de location d'une oeuvre protégée.

### E ###

- <span style="background: green">Emprunt</span> : action par laquelle un membre obtient une oeuvre temporairement.

- <span style="background: blue">emprunts</span> : répertoire contenant les oeuvres sous droit empruntées, chiffrées avec la clé du membre.

- <span style="background: green">Enrichissement des métadonées</span> : action des bibloithécaires consistant à compléter les informations descriptives d'une oeuvre.

- <span style="background: green">Etape de validation our rejet</span> : processus de décision des bibliothécaires lors de la modération d'une oeuvre.

- <span style="background: blue">Education</span> : sous-catégorie de livres.

### F ###

- <span style="background: blue">fond_commun</span> : répertoire contenant les oeuvres libres de droits mises à disposition par l'association.

- <span style="background: darkorange">Format Markdown</span> : format de fichier permettant le téléchargement d'oeuvres textuelles.

- <span style="background: darkorange">Format PDF</span> : format de fichier pour la numérisation des oeuvres.

### G ###

- <span style="background: green">Gérer les droits et copies</span> : action consistant à appliquer les règles légales de diffusion et d'exploitation.

### H ###

- <span style="background: blue">Histoire</span> : sous-catégorie de vidéos.

### J ###

- <span style="background: blue">Jazz</span> : sous-catégorie musicale.

- <span style="background: blue">Jeunesse</span> : sous-catégorie de livres.

### L ###

- <span style="background: green">Location</span> : action permettant à un membre de louer un oeuvre sous droit pour une durée limitée.

- <span style="background: blue">Livres</span> : catégorie d'oeuvres comprenant BD, romans, Jeunesse, Techniques, Education, Culture, Santé, etc.

### M ###

- <span style="background: blue">Membre</span> : toute personne inscrite disposant d'un compte sur l'application.

- <span style="background: blue">Métadonnées</span> : informations descriptives d'une oeuvre (auteur, éditeur, langue, date de publication, etc.).

- <span style="background: blue">Metal</span> : sous-catégorie musicale.

- <span style="background: blue">Musique</span> : catégorie d'oeuvres comprenant Classique, Jazz, Pop, Metal, etc.

### N ###

- <span style="background: green">Numériser</span> : action consistant à transformer une oeuvre physique en fichier numérique.

### O ###

- <span style="background: blue">Oeuvre numérique</span> : création humaine enregistrée dans un format informatique (texte, musique, vidéo, article).

- <span style="background: blue">Oeuvre sous droit</span> : création encore protégée par des ayants droit.

### P ###

- <span style="background: blue">Plateforme</span> : application centrale permettant la gestion, le partage et la modération des oeuvres.

- <span style="background: blue">Pop</span> : sous-catégorie musicale.

### R ###

- <span style="background: green">Reconnaissance de texte</span> : action permettant d'extraire du texte depuis une oeuvre numérisée via une IA.

- <span style="background: blue">Romans</span> : sous-catégorie de livres.

- <span style="background: blue">Répertoires</span> : métaphore utilisée pour représenter les différentes rubriques de stockage des oeuvres.

### S ###

- <span style="background: blue">Santé</span> : sous-catégorie de livres.

- <span style="background: blue">Séries</span> : sous-catégories de vidéos.

- <span style="background: blue">SF</span> : sous-catégorie de vidéos.

- <span style="background: blue">séquestre</span> : répertoire contenant les oeuvres sous droit en attente, gérées par l'association avec un accès restreint.

### T ###

- <span style="background: blue">Techniques</span> : sous-catégorie de livres.

- <span style="background: green">Téléchargement</span> : Action de récupérer une oeuvre en local depuis la bibliothèque numérique.
